# TABIT

韓国人旅行者向けの日本旅行AIプランナーです。

[한국어](./README.ko.md)

## プロジェクト概要

旅行者の予算、日程、同行者、興味、旅行スタイルを分析し、
条件に合った日本の旅行先と旅行プランを提案します。

## ターゲットユーザー

日本への個人旅行を計画している韓国人旅行者

## 主な機能

- ユーザー条件に合う旅行先を3件提案
- 旅行先ごとの推薦理由と予算を表示
- 日程別の旅行プランを生成
- 観光スポットと移動ルートを地図に表示
- 天候や休業日に応じた代替プランを提案

## 使用予定の技術

- Next.js
- TypeScript
- Python
- FastAPI
- PostgreSQL
- 生成AI API
- Maps API

## 開発方針

AIだけに判断を任せず、旅行データと推薦ロジックを組み合わせ、
根拠が明確で実現可能な旅行プランを提供します。

## 現在のプロトタイプ

`practice` フォルダには、HTML、CSS、JavaScriptで作成した旅行推薦プロトタイプがあります。

### 実装済み機能

- 予算、旅行日数、同行者タイプ、興味、旅行スタイルの入力
- 予算未入力および最低予算のチェック
- 興味が未選択の場合のチェック
- 興味に応じた旅行先の推薦
  - 自然: 札幌
  - グルメ: 福岡
  - ショッピング: 大阪
  - その他: 東京
- 推薦理由、想定予算、簡単な旅行日程の表示
- 旅行日数に応じた日程表示数の調整

### テスト項目

- 予算を入力しない場合、案内メッセージが表示されること
- 予算が30万ウォン未満の場合、案内メッセージが表示されること
- 興味を選択しない場合、案内メッセージが表示されること
- 自然を選択した場合、札幌が推薦されること
- グルメを選択した場合、福岡が推薦されること
- ショッピングを選択した場合、大阪が推薦されること
- 2日間の旅行を選択した場合、2日目までの日程のみ表示されること

### 現在の状態

HTML、CSS、JavaScriptのみを使用した一次プロトタイプを完成させました。
ユーザー入力をチェックし、興味に応じて旅行先を推薦し、
推薦結果と簡単な旅行日程を画面に表示します。

推薦ロジックと表示処理は関数として分離し、
旅行先データは配列で管理する形に整理しました。
