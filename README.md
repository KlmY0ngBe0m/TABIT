# TABIT

[한국어 README](./README.ko.md)

## プロジェクト紹介

TABITは、ユーザーの予算、旅行期間、旅行地域、人数、同行者、旅行スタイル、興味に合わせて、日本の旅行先と簡単な旅行日程を提案するAI旅行推薦Webサービスです。

最初はHTML/CSS/JavaScriptでプロトタイプを作成し、その後Next.jsとTypeScriptを使用したWebアプリケーションへ移行しました。現在はNext.js API RouteからOpenAI APIを呼び出し、ユーザー入力に基づいた旅行推薦を生成します。

## 対象ユーザー

日本旅行を計画している旅行者を対象としています。

特に、旅行先を自分で調べる時間を減らし、予算、日程、旅行地域、興味に合う候補を素早く確認したいユーザーを想定しています。

## 主な機能

- 予算、旅行期間、旅行地域、人数、同行者、旅行スタイル、興味の入力
- 日本地域マップUIによる希望旅行地域の選択
- 出発日と帰国日を基準にした旅行期間の自動計算
- 予算、日付、人数、旅行地域、興味の入力バリデーション
- 韓国語 / 日本語UI切り替え
- OpenAI APIによる旅行推薦生成
- 選択した旅行条件の確認機能
- APIリクエスト中のローディング表示
- API失敗時のfallback推薦
- 推薦旅行先、推薦理由、簡単な日程の表示
- Supabaseを利用した推薦記録の保存
- 推薦記録に基づく統計API
- 管理者ログイン / ログアウト
- 管理者専用の推薦記録統計ダッシュボード

## 現在の処理フロー

    ユーザー入力
    -> React state管理
    -> 入力値バリデーション
    -> 選択した旅行条件の確認
    -> Next.js API Route
    -> OpenAI APIリクエスト
    -> AI旅行推薦生成
    -> Supabaseに推薦記録を保存
    -> 結果カード表示

OpenAI APIキーが設定されていない場合、またはAPIリクエストに失敗した場合は、選択された興味を基準にfallback推薦を返します。

管理者は別途ログインページから推薦記録の統計ダッシュボードにアクセスできます。

    管理者ログイン
    -> 管理者Cookie発行
    -> 推薦記録統計APIリクエスト
    -> Supabase推薦記録取得
    -> 統計ダッシュボード表示

## テスト項目

- 予算が入力されていない場合にエラーメッセージが表示されるか
- 最低予算未満の場合にエラーメッセージが表示されるか
- 出発日と帰国日が正しく設定されるか
- 帰国日が出発日より早い場合にエラーメッセージが表示されるか
- 旅行期間が自動で計算されるか
- 人数が1人の場合に同行者選択が非表示になるか
- 旅行地域が選択されていない場合にエラーメッセージが表示されるか
- 興味が選択されていない場合にエラーメッセージが表示されるか
- 推薦生成前に選択した旅行条件の確認画面が表示されるか
- 推薦生成中にローディング表示が出るか
- OpenAI APIが成功した場合にAI推薦結果が表示されるか
- OpenAI APIが失敗した場合にfallback推薦が表示されるか
- 韓国語 / 日本語UI切り替えができるか
- PC / モバイル画面で地域選択UIが正常に表示されるか
- 推薦結果がSupabaseに保存されるか
- `/api/stats` で推薦記録統計が返されるか
- 管理者ログイン成功時に `/admin/stats` にアクセスできるか
- 管理者ログインなしで `/admin/stats` にアクセスした場合、ログインページへ移動するか
- 管理者ログアウト後に統計ページへのアクセスが制限されるか

## 環境変数

`frontend/.env.local` ファイルを作成し、以下の環境変数を設定します。

    OPENAI_API_KEY=your_openai_api_key_here
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
    SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
    ADMIN_PASSWORD=your_admin_password
    NEXT_PUBLIC_BASE_URL=http://localhost:3000

`.env.local` はGitHubにコミットしてはいけません。

## 管理者機能

管理者は以下のパスからログインできます。

    /admin/login

ログイン後、推薦記録統計ダッシュボードで保存された推薦データを確認できます。

    /admin/stats

管理者ダッシュボードでは、以下の情報を確認できます。

- 総推薦数
- 平均旅行期間
- 平均人数
- 推薦都市 TOP 5
- 選択地域 TOP 5
- 興味 TOP 5
- 旅行スタイル TOP 5

## 現在の状態

OpenAI APIを利用したAI旅行推薦機能と、Supabaseベースの推薦記録保存機能を実装しました。

現在は、ユーザーが入力した旅行条件をもとに日本の旅行先と簡単な日程を推薦し、推薦結果をSupabaseに保存します。また、管理者ログイン機能と推薦記録統計ダッシュボードを通じて、保存されたデータを分析できます。
