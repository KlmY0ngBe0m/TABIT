# TABIT

韓国人旅行者向けの日本旅行AIプランナーです。

[한국어](./README.ko.md)

## プロジェクト概要

旅行者の予算、日程、同行者、興味、旅行スタイルを分析し、
条件に合った日本の旅行先と旅行プランを提案します。

## ターゲットユーザー

日本への個人旅行を計画している韓国人旅行者

## 主な機能

- ユーザー条件に合う旅行先を3件提案
- 旅行先ごとの推薦理由と予算を表示
- 日程別の旅行プランを生成
- 観光スポットと移動ルートを地図に表示
- 天候や休業日に応じた代替プランを提案

## 使用予定の技術

- Next.js
- TypeScript
- Python
- FastAPI
- PostgreSQL
- 生成AI API
- Maps API

## 開発方針

AIだけに判断を任せず、旅行データと推薦ロジックを組み合わせ、
根拠が明確で実現可能な旅行プランを提供します。
