# TABIT

한국인 여행자를 위한 일본 여행 AI 플래너입니다.

[日本語](./README.md)

## 프로젝트 소개v
여행자의 예산, 일정, 동행자, 관심사와 여행 스타일을 분석하여
조건에 맞는 일본 여행지와 여행 계획을 제안합니다.

## 대상 사용자

일본 자유여행을 계획하는 한국인 여행자

## 주요 기능

- 사용자 조건에 맞는 여행지 3곳 추천
- 여행지별 추천 이유와 예상 예산 표시
- 일자별 여행 일정 생성
- 관광지와 이동 경로를 지도에 표시
- 날씨와 휴무일에 따른 대체 일정 제안

## 사용 예정 기술

- Next.js
- TypeScript
- Python
- FastAPI
- PostgreSQL
- 생성형 AI API
- 지도 API

## 개발 방향

AI의 판단에만 의존하지 않고 여행 데이터와 추천 로직을 결합하여,
추천 근거가 명확하고 실제로 실행 가능한 여행 계획을 제공합니다.

## 현재 프로토타입

`practice` 폴더에는 HTML, CSS, JavaScript로 만든 여행 추천 프로토타입이 있습니다.

### 구현된 기능

- 예산, 여행 기간, 동행 유형, 관심사, 여행 스타일 입력
- 예산 미입력 및 최소 예산 검증
- 관심사 미선택 검증
- 관심사에 따른 추천 여행지 표시
  - 자연: 삿포로
  - 맛집: 후쿠오카
  - 쇼핑: 오사카
  - 기타: 도쿄
- 추천 이유, 예상 예산, 간단 일정 표시
- 여행 기간에 따라 일정 표시 개수 조정

### 테스트 항목

- 예산을 입력하지 않았을 때 안내 문구가 표시되는지
- 예산이 30만 원 미만일 때 안내 문구가 표시되는지
- 관심사를 선택하지 않았을 때 안내 문구가 표시되는지
- 자연 선택 시 삿포로가 추천되는지
- 맛집 선택 시 후쿠오카가 추천되는지
- 쇼핑 선택 시 오사카가 추천되는지
- 2일 여행 선택 시 2일차 일정까지만 표시되는지
